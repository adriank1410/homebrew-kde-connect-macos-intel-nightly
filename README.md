# Adriank1410 Kde-nightly

## How do I install these formulae?

`brew install adriank1410/kde-nightly/<formula>`

Or `brew tap adriank1410/kde-nightly` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "adriank1410/kde-nightly"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
